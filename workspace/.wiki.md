# Project Summary
**RE:MOV3** is an innovative online platform offering specialized physiotherapy services, focusing on rehabilitation, strength training, and professional development. The project aims to enhance user experience through an intuitive interface, seamless service booking, and increased user engagement with vibrant design elements and structured layouts.

# Project Module Description
The project consists of the following functional modules:
- **Landing Page**: Features a redesigned layout with a sticky navbar, video background, and sections for "Who I Am", Services, Methodology, Testimonials, and Contact.
- **Language Support**: Allows users to toggle between Spanish and Catalan.
- **Contact Integration**: Provides direct links for WhatsApp and email, along with a booking button for services.
- **Footer**: Displays contact information and social media links.

# Usage
To get started with the project, follow these steps:
1. **Install Dependencies**: Run the command to install all necessary packages.
   ```
   pnpm install
   ```
2. **Build the Project**: Compile the project for production.
   ```
   pnpm run build
   ```
3. **Run Linting**: Ensure code quality by running lint checks.
   ```
   pnpm run lint
   ```
